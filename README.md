# WebRTC Meet the Strangers

Audio and video chat with Javascript

### Author: AmiraliShamlo

\
Link - http://localhost:3000

## Installation Nodejs

```
npm init
```

```
npm install --save express
```

```
npm install --save socket.io
```

## Installation nodemon

```
npm install nodemon
```

## How to use

```
npm install
```

```
npm start
```
